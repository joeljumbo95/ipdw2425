Style et Thème

option de présentation : thème : light or dark or grey

option de présentation : font : 3 font colors

Boutons afin de mettre différents types de background colors entre blanc, gris et foncé. La logique pour que les boutons fonctionnent est dans la fonction ctrl_head() et est récupérée dans le style du html_head.

Login

sauvegarde de l'identification (login) : file

Le fichier de l'identification se trouve dans asset/database/login.csv et reprend les différentes personnes autorisées.

Voici le nom des fonctions qui permettent le fonctionnement du login :

  • function login_validate($input, $password)
  • function html_open_form()
  • function html_close_form()
  • function html_unidentified_user()
  • function html_link_home()
  • function main_login()
